Angular2-Draggable

angular2-draggable is an Angular(ver >= 4.x) directive that makes the DOM element draggable

Documents Github

Installation

```bash # if you use npm npm install angular2-draggable --save # or if you use yarn yarn add angular2-draggable ```

Import

```typescript import {{'{'}} AngularDraggableModule } from 'angular2-draggable'; @NgModule({{'{'}} imports: [ ..., AngularDraggableModule ], }) export class AppModule {{'{'}} } ```
